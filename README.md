# Kurulum

```bash
docker compose -d --build
```
Komutu ile Docker container build edildikten sonra
web sunucusu ve mssql veritabanı sunucusu için iki ayrı container oluşacak.

Web sunucusu açıldıktan sonra lütfen
```bash
http://localhost:8080/includes/init.php
```
sayfasına gidip tabloları ve içindeki verileri veritabanına aktarın.
